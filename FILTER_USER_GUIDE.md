# Filter & Search User Guide

## Overview

The Monday.com Vacation Dashboard now includes powerful filtering and search capabilities to help you quickly find and analyze vacation data.

## Filter Bar Location

The filter bar appears at the top of the dashboard, just below the header and above the statistics cards.

## Available Filters

### 1. 🔍 Search Employee Name

**What it does**: Searches for employees by name in real-time

**How to use**:
1. Click in the "Search Employee" text box
2. Start typing an employee's name
3. Results update automatically as you type
4. Search is case-insensitive (works with uppercase or lowercase)

**Example**: Type "john" to find "John Smith", "Johnny Doe", etc.

---

### 2. 👥 Filter by Manager

**What it does**: Shows only vacation requests for selected manager(s)

**How to use**:
1. Click the "Manager" dropdown
2. Select one or more managers from the list
3. Selected managers appear as colored tags below the dropdown
4. Click the ❌ on any tag to remove that manager from the filter
5. Select "All Managers" to clear the filter

**Tips**:
- You can select multiple managers at once
- The dropdown shows all unique managers from your data
- Managers are sorted alphabetically

---

### 3. ✅ Filter by Status

**What it does**: Shows only vacation requests with selected status(es)

**How to use**:
1. Click the "Status" dropdown
2. Select one or more statuses:
   - **Approved**: Confirmed vacation requests
   - **Pending**: Awaiting approval
   - **Rejected**: Denied requests
   - **Unknown**: Status not set
3. Selected statuses appear as colored tags below the dropdown
4. Click the ❌ on any tag to remove that status from the filter
5. Select "All Statuses" to clear the filter

**Tips**:
- You can select multiple statuses at once
- Useful for focusing on pending requests or approved vacations

---

### 4. 📅 Filter by Date Range

**What it does**: Shows only vacation requests within a specific date range

**How to use**:
1. **From Date**: Click the "From" date picker and select a start date
2. **To Date**: Click the "To" date picker and select an end date
3. The system will show vacations that overlap with your selected range
4. You can set just a start date, just an end date, or both

**Tips**:
- Leave both dates empty to see all vacations
- Set only "From" to see all future vacations from that date
- Set only "To" to see all vacations up to that date
- The "To" date must be after the "From" date

---

## Combining Filters

**All filters work together!**

You can use multiple filters at the same time to narrow down your results:

**Example 1**: Find John's pending requests
- Search: "john"
- Status: "Pending"

**Example 2**: See all approved vacations for a specific manager in Q2
- Manager: "Jane Doe"
- Status: "Approved"
- Date Range: April 1 - June 30

**Example 3**: Find all upcoming vacations
- Date Range: Today onwards
- Status: "Approved"

## Filter Indicators

### Active Filter Badge
- Shows the number of active filters (e.g., "3 active")
- Appears in the filter bar header
- Helps you see at a glance how many filters are applied

### Results Counter
- Shows "Showing X of Y results"
- Appears below the filter bar when filters are active
- Helps you understand how much data is being filtered

### Selected Tags
- Colored tags show which managers/statuses are selected
- Each tag has an ❌ button to remove it individually
- Makes it easy to see and manage your active filters

---

## Clearing Filters

### Clear All Filters
1. Click the red "Clear All" button in the filter bar header
2. All filters reset to default (showing all data)
3. The button only appears when filters are active

### Clear Individual Filters
- **Search**: Delete text from the search box
- **Manager/Status**: Click ❌ on individual tags or select "All" in dropdown
- **Date Range**: Clear the date inputs

## Tips & Best Practices

### 🎯 Quick Searches
- Use the search box for quick employee lookups
- Combine with status filter to find specific requests

### 📊 Manager Reports
- Filter by manager to see their team's vacation patterns
- Use date range to analyze specific periods

### 📅 Planning Ahead
- Use forecast view with date range to plan coverage
- Filter by approved status to see confirmed vacations

### 🔄 Real-Time Updates
- Filters update instantly as you type or select
- No need to click "Apply" or "Search" buttons
- Data refreshes automatically

### 💡 Performance
- Search has a small delay (300ms) to improve performance
- This prevents excessive filtering while you type
- You won't notice the delay in normal use

## Troubleshooting

### No Results Showing?
- Check if you have filters applied (look for the active filter badge)
- Click "Clear All" to reset and see all data
- Verify your date range isn't too restrictive

### Filter Not Working?
- Refresh the page (Ctrl+R or Cmd+R)
- Check that the dashboard has loaded data
- Try clearing all filters and applying them again

### Dropdown Not Showing Options?
- Make sure data has loaded (check stats cards)
- Refresh the dashboard data using the Refresh button
- Check your internet connection
